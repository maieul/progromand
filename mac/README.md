progromand
==========

Un clavier romand avec des accès facilités aux touches de programmation

# Accolades et crochet
Comme pour le clavier français :
* alt + 8 => {
* alt + 9 => }
* alt + maj + 8 => [
* alt + maj + 9 => ]

# Autres programmation :
- alt + l => |
- alt + y => /
- alt + maj + y => \
- alt + d => #
- alt + > => =
- alt + x => *
- alt + maj + x => +
- maj + $ => "

# Autres français
- alt + c => ç
- alt + u => ù
- alt + e -> ê
- alt + $ => €
- alt + § => ¶ 
- alt + i => î
